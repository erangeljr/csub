1. What is the purpose of the branch delay slot?

2. The following questions pertains to what can or cannot be placed in a
   branch delay slot.

          branch delay slot.
       B. Give examples of instructions that CANNOT be placed in a
          branch delay slot.

          move $t1, $zero

      ...more code ...
