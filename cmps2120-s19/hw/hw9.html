The Basics of Counting

Express large numbers in exponential or factorial notation (e.g. 246 , 14!).
1. There are four major auto routes from Boston to Detroit and six from Detroit to Los Angeles. How many major auto routes are there from Boston to Los Angeles via Detroit?

2. How many bit strings are there of length six or less? (include the empty string)

For the next questions, assume set S = {1,2,3,...999}. How many numbers in S are

3. divisible by 7?

4. divisible by 7 but not by 11?

5. divisible by both 7 and 11?

6. divisible by either 7 or 11 ?

7. divisible by exactly one of 7 and 11? (this is XOR so exclude twice)

8. divisible by neither 7 nor 11 ?

9. From above... How many numbers in S are have distinct digits? (this excludes numbers such as 11,22,33,...101,etc.) Hint: fix the leftmost digit first since it cannot be zero.



*Beware - this problem is tricky.

10. From above... How many numbers in S have distinct digits and are even? Hint: it is easier to count odds since zero is not a final digit in an odd. For 3 digit numbers, start by fixing the rightmost digit, then fix the leftmost digit, then finally fix the middle digit.

11. How many license plates can be made using either three digits followed by three letters or three letters followed by three digits?



The next questions count various strings from the set S of English letters, where the cardinality of S is 26.

12. How many strings of size 8 if letters can be repeated?

13. How many strings of size 8 if no letter can be repeated?

14. How many strings of size 8 that start with X, if letters can be repeated?

15. How many strings of size 8 start with X, if no letter can be repeated?

16. How many strings of size 8 start and end with X, if leters can be repeated?

17. How many different functions are there from a set with 10 elements to a set of 2 elements? to a set of 3 elements?

18. How many subsets of a set with 100 elements have more than one element?



The next problems involve counting permutations. Express your answer in permutation form. In how many ways can a photographer at a wedding arrange 6 people in a row from a group of 10 people, where the bride and the groom are among these 10 people, if

19. the bride must be in the picture?

20. both the bride and the groom must be in the picture?

21. exactly one of the bride and the groom is in the picture?

22. either the bride or groom but not both ?



23. Let S = {2,4,6,8,10,12}. How many subsets of S contain elements whose sum is at most 18 (the classic knapsack problem) if you include the empty knapsack? How many potential knapsacks are there?
