Give the name of the syllogism used in these arguments and state whether the argument is valid or invalid.

1. If an animal is a bird then it has wings. Pooh doesn't have wings so Pooh is not a bird.

2. Roos live in Austrialia and roos are marsupials. Roos are marsupials.

3. If Linda is an excellent swimmer then she can work as a lifeguard. Linda is an excellent swimmer. I conclude she can work as a lifeguard.

4. If x^2 is irrational, then x is irrational. Since x is irrational, we conclude that x^2 must be irrational.
